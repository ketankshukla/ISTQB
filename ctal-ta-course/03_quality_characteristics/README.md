## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Describe the ISO 25010 quality characteristics and their sub-characteristics
2. Identify appropriate test types for each quality characteristic
3. Select suitable test design techniques for testing quality characteristics
4. Describe security testing approaches and common vulnerability types
5. Describe reliability testing including fault tolerance and recoverability
6. Describe efficiency testing including time behavior and resource utilization
7. Describe maintainability and portability testing approaches

---

## ⭐ Key Concepts to Master

- The 8 ISO 25010 quality characteristics: Accuracy, Suitability, Interoperability, Security, Reliability, Efficiency, Maintainability, Portability
- Sub-characteristics within each (e.g., Security: confidentiality, integrity, availability)
- Test types appropriate for each characteristic (functional, non-functional)
- Techniques for security testing (penetration testing, vulnerability scanning)
- Techniques for reliability testing (stress testing, failover testing)
- Techniques for efficiency testing (load testing, performance testing)
- The difference between functional and non-functional testing

---

## 🎓 Exam Tips

- Expect scenario-based questions asking you to select the appropriate quality characteristic to test
- Security, Reliability, and Efficiency are the most heavily tested characteristics
- Know the difference between sub-characteristics (e.g., confidentiality vs. integrity)
- Understand which test types map to which characteristics
- Know common tools for each type of testing
