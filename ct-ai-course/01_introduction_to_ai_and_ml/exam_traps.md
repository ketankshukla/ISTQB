# 🪤 Chapter 1 — Introduction to AI and ML: Exam Traps

## 🪤 Trap 1: Confusing AI, ML, and Deep Learning

**Trap:** All ML is AI, but not all AI is ML. All DL is ML, but not all ML uses DL.

**Example question:**
> Which statement is TRUE?
>
> A. All AI systems use machine learning
> B. All machine learning systems use deep learning
> C. All deep learning systems use neural networks
> D. Expert systems are a type of machine learning

**Correct answer:** C — Deep learning is a subset of ML that uses neural networks with many layers. Expert systems (D) use rules, not ML. Not all AI uses ML (A). Not all ML uses DL (B).

---

## 🪤 Trap 2: Thinking Current AI Is General AI

**Trap:** All practical AI today is Narrow AI. General AI and Super AI do not exist yet.

**Example question:**
> A virtual assistant like Siri is an example of:
>
> A. General AI
> B. Super AI
> C. Narrow AI
> D. Artificial Superintelligence

**Correct answer:** C — Siri is Narrow AI. It performs specific tasks (voice commands, searches) and cannot generalize to any task.

---

## 🪤 Trap 3: Confusing Supervised and Unsupervised Learning

**Trap:** Supervised learning uses labeled data. Unsupervised learning uses unlabeled data.

**Example question:**
> A system groups customers based on purchasing behavior without predefined categories. This is:
>
> A. Supervised learning
> B. Unsupervised learning
> C. Reinforcement learning
> D. Deep learning

**Correct answer:** B — Grouping without predefined categories = clustering = unsupervised learning.

---

## 🪤 Trap 4: Thinking RPA Is AI

**Trap:** RPA follows predefined rules. It is not AI unless combined with ML/NLP for decision-making.

**Example question:**
> A software robot that copies data from one spreadsheet to another following exact rules is:
>
> A. An example of machine learning
> B. Robotic Process Automation (RPA), not true AI
> C. A neural network
> D. An expert system

**Correct answer:** B — RPA follows rules. It does not learn from data, so it is not ML/AI.

---

## 🪤 Trap 5: Confusing Classification and Regression

**Trap:** Classification predicts discrete categories. Regression predicts continuous values.

**Example question:**
> Predicting the price of a house based on its features is:
>
> A. Classification
> B. Regression
> C. Clustering
> D. Reinforcement learning

**Correct answer:** B — Price is a continuous numerical value, so this is regression.
