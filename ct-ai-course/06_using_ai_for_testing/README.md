# Chapter 6 — Using AI for Testing

## Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** AI-powered test generation, test analysis, defect prediction, autonomous testing, tool selection

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Explain how AI can be used to generate tests
2. Describe AI-powered test analysis and optimization
3. Explain defect prediction using AI
4. Describe visual testing with AI
5. Explain autonomous testing concepts
6. Describe considerations for selecting and evaluating AI testing tools

---

## Key Concepts

- AI test generation: requirements-based, code-based, model-based
- Intelligent test prioritization and selection
- Defect prediction and root cause analysis
- Visual regression testing with computer vision
- Autonomous testing
- Tool selection and validation for AI testing tools
