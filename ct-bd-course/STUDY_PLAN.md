## Study Techniques

**Active Recall**
- After each lesson, close the file and write down 3 key takeaways
- Explain concepts aloud as if teaching someone else

**Spaced Repetition**
- Revisit exam traps from previous chapters every 3 days
- Quiz yourself on glossary terms daily

**Question Analysis**
- For every wrong practice answer, write why the correct answer is right
- Identify the syllabus topic each question maps to

**Timed Practice**
- Practice questions: 1.5 minutes per question
- Full mock exams: 60 minutes strictly timed
