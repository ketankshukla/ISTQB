# Chapter 6 — DevOps Quality Tools and Automation

## Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K1, K2
- **Focus:** Pipeline tools, test automation frameworks, reporting, tool selection

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Identify CI/CD platform tools
2. Describe test automation frameworks for DevOps
3. Explain test result reporting and dashboards
4. Select appropriate tools for DevOps quality scenarios
5. Integrate quality tools into pipelines
6. Manage test data and environments in DevOps

---

## Key Concepts

- CI/CD platforms (Jenkins, GitLab CI, GitHub Actions, Azure DevOps)
- Test automation frameworks
- Test result reporting and dashboards
- Test data management in pipelines
- Environment provisioning tools
- Tool selection criteria
