# 🚀 Chapter 2 — CI/CD Pipeline Testing

## 🎓 Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2, K3
- **Focus:** Pipeline stages, quality gates, test automation in CI/CD, deployment strategies

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Describe CI/CD pipeline stages and testing activities
2. Design and implement quality gates
3. Select appropriate test types for each pipeline stage
4. Explain deployment strategies (blue-green, canary, feature flags)
5. Test rollback and recovery mechanisms

---

## ⭐ Key Concepts

- CI/CD pipeline stages
- Quality gates and criteria
- Test pyramid in pipelines
- Deployment strategies
- Rollback and recovery testing
- Pipeline as code
