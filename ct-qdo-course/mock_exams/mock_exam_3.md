### Q1-Q10

2. CI/CD pipeline unit tests: A) After acceptance B) First for fast feedback C) Manual only D) Skip — **First for fast feedback**
3. Quality gate: A) Slows unnecessarily B) Enforces criteria C) Optional D) Replaces testing — **Enforces criteria**

5. Canary: A) 100% immediately B) Subset with monitoring C) No monitoring D) After 24h — **Subset with monitoring**
6. Feature flags: A) Deploy without test B) Control visibility C) Only A/B D) Auto rollback — **Control visibility**
7. Rollback testing: A) Only success B) Verify safe restore C) Only migrations D) Only UI — **Verify safe restore**
8. SAST runs: A) Production B) Early pipeline C) After deploy D) Manual only — **Early pipeline**
