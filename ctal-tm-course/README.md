# 📚 ISTQB CTAL-TM — Advanced Test Manager Self-Study Course

A complete, exam-ready self-study course for the ISTQB CTAL-TM (Certified Tester Advanced Level — Test Manager) certification.

---

## 🎓 Exam Facts

| Parameter | Value |
|-----------|-------|
| Full name | ISTQB Certified Tester Advanced Level — Test Manager |
| Questions | 65 multiple-choice |
| Duration | 180 minutes |
| Pass mark | 65% (42 out of 65) |
| Closed book | Yes |
| Syllabus | CTAL-TM v4.0 |
| Prerequisite | ISTQB CTFL Foundation Level (any version) |

## ⭐ How to Use This Course

1. **Start with `STUDY_PLAN.md`** — pick the 4-week or 5-week track based on your schedule.
2. **Work through chapters in order** — read `lessons.md`, then `exam_traps.md`, then attempt `practice_questions.md`.
3. **Complete case studies** after finishing all chapters.
4. **Take mock exams** under timed conditions (closed book, 180 minutes).
5. **Review `EXAM_STRATEGY.md`** the day before your exam.
