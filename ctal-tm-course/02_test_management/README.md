# Chapter 2 — Test Management

## Exam Weighting

- **Estimated weight:** ~45% (~29 questions out of 65)
- **K-levels:** K2 (understand), K3 (apply), K4 (analyze)
- **Focus:** Test organization, estimation, risk-based testing, defect management, test metrics, test team management

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Describe different test organization structures (independent, integrated, matrix)
2. Apply test estimation techniques (bottom-up, top-down, Wideband Delphi)
3. Explain risk-based testing and prioritize tests based on risk
4. Describe defect management processes and metrics
5. Explain test metrics and their use in management decisions
6. Describe the test manager's role in managing the test team

---

## Key Concepts

- Test organization: independent, integrated, matrix-based
- Estimation: bottom-up, top-down, Wideband Delphi, experience-based
- Risk-based testing: risk identification, assessment, mitigation, prioritization
- Defect management: lifecycle, classification, metrics, reporting
- Test metrics: coverage, defect, cost, progress metrics
- Test team management: skills, motivation, communication, leadership
