# Chapter 1 — Testing Process

## Exam Weighting

- **Estimated weight:** ~20% (~13 questions out of 65)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Test planning, monitoring, control, test analysis and design, test implementation and execution

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Explain the test planning activities and their outputs
2. Describe test monitoring and control activities
3. Explain test analysis and design activities
4. Describe test implementation, execution, and completion activities
5. Apply appropriate entry and exit criteria for test levels

---

## Key Concepts

- Test planning: objectives, scope, approach, resources, schedule
- Test monitoring: tracking progress, collecting metrics, reporting status
- Test control: taking corrective action based on monitoring results
- Entry criteria and exit criteria
- Test analysis, design, implementation, execution, completion
- Test estimation techniques
- Test strategy and test approach
