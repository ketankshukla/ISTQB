# 🔎 Chapter 3 — Reviews

## 🎓 Exam Weighting

- **Estimated weight:** ~10% (~6 questions out of 65)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Review types, review process, test manager's role in reviews, review metrics

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Describe different types of reviews (informal, walkthrough, technical review, inspection)
2. Explain the review process stages
3. Describe the test manager's role in planning and managing reviews
4. Explain review metrics and their use
5. Describe factors for successful reviews

---

## ⭐ Key Concepts

- Review types: informal review, walkthrough, technical review, inspection
- Review process: planning, preparation, review meeting, rework, follow-up
- Roles: moderator, author, reviewer, scribe
- Entry and exit criteria for reviews
- Review metrics: defect density, review efficiency, checklists
