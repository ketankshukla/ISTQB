### 🔷 Q15 (TM-4.3, K2)

C. The programming language used by developers

---

## ✔️ Answers and Rationales

### 🔷 Q1
**Answer: B**

**Rationale:** Test management tools manage test cases, defects, and traceability. Performance, static analysis, and security tools have different purposes.

### 🔷 Q2
**Answer: B**

**Rationale:** Daily regression tests are repetitive, stable, and frequent — ideal for automation. One-time tests, usability tests, and unstable requirements are poor candidates.

### 🔷 Q3
**Answer: C**

**Rationale:** TCO = $20,000 + $8,000 + $12,000 + ($4,000 × 2) = $48,000. Purchase, training, implementation, and 2 years of maintenance.

### 🔷 Q4
**Answer: B**

**Rationale:** SonarQube analyzes code for bugs, vulnerabilities, and code smells without executing it — a static analysis tool.

### 🔷 Q5
**Answer: B**

**Rationale:** Capture/replay scripts are tightly coupled to the UI. Any UI change breaks them, making them brittle.

### 🔷 Q6
**Answer: B**

**Rationale:** Year 1 costs exceed benefits due to framework development. ROI becomes positive in later years.

### 🔷 Q7
**Answer: B**

**Rationale:** Requirements definition is the first step. Demos, purchases, and preferences come after requirements are clear.

### 🔷 Q8
**Answer: B**

**Rationale:** Automation adds value for repetitive, stable tests. It cannot replace all manual testing and requires maintenance.

### 🔷 Q9
**Answer: B**

**Rationale:** PoC validates the tool solves the real problem before committing to purchase and deployment.

### 🔷 Q10
**Answer: B**

**Rationale:** Consistent pass rates (not flaky tests) indicate a healthy framework. Number of tests, lines of code, and licenses are poor health indicators.

### 🔷 Q11
**Answer: B**

**Rationale:** JMeter is an open-source performance and load testing tool.

### 🔷 Q12
**Answer: B**

**Rationale:** POM separates UI element locators from test logic, making tests more maintainable when the UI changes.

### 🔷 Q13
**Answer: B**

**Rationale:** ROI = (Benefits - Costs) / Costs. This measures return relative to the investment made.

### 🔷 Q14
**Answer: B**

**Rationale:** A pilot validates the tool in a real context, identifies issues, and gathers lessons before full deployment.

### 🔷 Q15
**Answer: B**

**Rationale:** Execution frequency and test stability are the most important factors. Frequently executed, stable tests provide the best automation ROI.
