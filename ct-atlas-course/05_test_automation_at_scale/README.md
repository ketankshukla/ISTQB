# Chapter 5 — Test Automation at Scale

## Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2, K3
- **Focus:** Shared frameworks, cross-team reuse, CI/CD at scale, automation governance

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Design shared automation frameworks for multiple teams
2. Coordinate cross-team test automation
3. Manage test data at scale
4. Integrate automation into CI/CD pipelines across teams
5. Govern automation standards and practices
6. Measure and optimize automation ROI at scale

---

## Key Concepts

- Shared automation frameworks
- Cross-team reuse
- Test data management at scale
- CI/CD pipeline orchestration
- Automation governance
- ROI measurement at scale
