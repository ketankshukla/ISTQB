# Chapter 4 — Quality Metrics and Reporting

## Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2, K3
- **Focus:** Metrics selection, dashboards, KPIs, feedback loops, reporting cadence

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Select appropriate quality metrics for different audiences
2. Design effective quality dashboards
3. Distinguish leading and lagging indicators
4. Use metrics to drive improvement
5. Avoid metric misuse and gaming
6. Report quality status at team, program, and portfolio levels

---

## Key Concepts

- Leading vs lagging indicators
- DORA metrics
- Test metrics (coverage, pass rate, execution time)
- Defect metrics (density, escape rate, MTTR)
- Dashboard design principles
- Metric anti-patterns
