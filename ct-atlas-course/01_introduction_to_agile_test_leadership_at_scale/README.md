# 🌟 Chapter 1 — Introduction to Agile Test Leadership at Scale

- **Estimated weight:** ~10% (~4 questions out of 40)

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Define agile at scale
2. Identify challenges of testing in large agile organizations
3. Describe common scaling frameworks (SAFe, LeSS, Spotify)
4. Explain the role of test leadership at scale
5. Identify coordination needs across teams
