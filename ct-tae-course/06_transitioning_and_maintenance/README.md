# ⚙️ Chapter 6 — Transitioning and Maintaining Test Automation

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Transition strategies, maintenance types, legacy automation, retirement criteria

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Explain strategies for transitioning from manual to automated testing
2. Describe types of maintenance for a TAS
3. Identify when to retire tests or a TAS
4. Explain how to manage legacy automation
5. Describe the skills and roles needed for automation maintenance

---

## ⭐ Key Concepts

- Transition strategies: manual → automated, legacy → new
- Maintenance types: corrective, adaptive, perfective
- Test and TAS retirement criteria
- Legacy automation management
- Team skills and responsibilities for maintenance
