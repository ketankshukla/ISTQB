# ⚙️ Chapter 2 — Preparing for Test Automation

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Tool evaluation, PoC/PoV, requirements analysis, risk assessment, team skills

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Explain how to evaluate and select test automation tools
2. Describe the purpose and process of a Proof of Concept (PoC) and Proof of Value (PoV)
3. Identify requirements for a test automation solution
4. Assess risks associated with test automation
5. Identify the skills needed for a test automation team

---

## ⭐ Key Concepts

- Tool evaluation criteria
- PoC vs. PoV
- TCO (Total Cost of Ownership)
- Automation requirements analysis
- Risk identification and mitigation
- Team skill assessment
