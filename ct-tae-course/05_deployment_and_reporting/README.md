# 📝 Chapter 5 — Deployment and Reporting

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Deployment strategies, metrics, dashboards, ROI, effectiveness measurement

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Explain deployment strategies for test automation (big-bang, incremental, pilot)
2. Identify and calculate test automation metrics
3. Design dashboards for different stakeholders
4. Explain ROI and effectiveness measurement
5. Describe reporting practices for test automation

---

## ⭐ Key Concepts

- Deployment strategies: big-bang, incremental, pilot
- Metrics: coverage, effectiveness, efficiency, flakiness
- Dashboards for testers, developers, management
- ROI calculation and business value
- Reporting types and audiences
