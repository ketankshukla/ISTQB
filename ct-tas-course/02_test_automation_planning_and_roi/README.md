# Chapter 2 — Test Automation Planning and ROI

## Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2, K3
- **Focus:** Business case, cost-benefit, risk assessment, scope prioritization

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Create a business case for automation
2. Calculate ROI for automation investment
3. Assess risks and constraints
4. Prioritize tests for automation
5. Define automation scope and objectives
6. Plan resources and timelines

---

## Key Concepts

- Business case components
- ROI calculation methods
- TCO (Total Cost of Ownership)
- Risk assessment
- Test selection criteria
- Resource planning
