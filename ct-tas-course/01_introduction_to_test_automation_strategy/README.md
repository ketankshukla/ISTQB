# Chapter 1 — Introduction to Test Automation Strategy

## Exam Weighting

- **Estimated weight:** ~10% (~4 questions out of 40)
- **K-levels:** K1, K2
- **Focus:** Automation definition, levels, scope, strategy vs execution

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Define test automation strategy
2. Distinguish test automation from automated testing
3. Identify levels of automation (unit, integration, system, acceptance)
4. Explain the scope and objectives of automation
5. Describe the role of strategy in automation success

---

## Key Concepts

- Test automation strategy definition
- Strategy vs execution
- Automation levels (test pyramid)
- Scope and objectives
- Success factors
