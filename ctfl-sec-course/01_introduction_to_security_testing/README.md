# 🌟 Chapter 1 — Introduction to Security Testing

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~3 questions out of 20)
- **K-levels:** K1, K2
- **Focus:** What is security testing, why it matters, key concepts

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Define security testing
2. Explain the CIA triad
3. Identify common security threats
4. Describe the relationship between threats, vulnerabilities, and risks
5. Recognize when security testing is needed

- Security testing definition
- Confidentiality, Integrity, Availability (CIA)
- Threats, vulnerabilities, risks
