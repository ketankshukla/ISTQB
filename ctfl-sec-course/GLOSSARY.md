# 📖 CTFL-SEC Glossary

## ⭐ A

- **Authentication** — Verifying identity of a user or system
- **Authorization** — Determining access rights of an authenticated user

## ⭐ B

- **Black-box testing** — Testing without knowledge of internal structure

## ⭐ C

- **CIA triad** — Confidentiality, Integrity, Availability
- **Confidentiality** — Preventing unauthorized access to information
- **Cross-site scripting (XSS)** — Injection of malicious scripts into web pages

## ⭐ D

- **Denial of service (DoS)** — Attack that makes a resource unavailable
- **DAST (Dynamic Application Security Testing)** — Testing running application for vulnerabilities

## ⭐ I

- **Integrity** — Preventing unauthorized modification of information
- **Injection** — Attack that sends malicious data as input

## ⭐ M

- **Malware** — Malicious software designed to damage or gain unauthorized access

## ⭐ O

- **OWASP** — Open Web Application Security Project
- **OWASP Top 10** — Standard awareness document for web application security risks

## ⭐ P

- **Penetration testing** — Simulated attack to identify vulnerabilities
- **Phishing** — Fraudulent attempt to obtain sensitive information

## ⭐ R

- **Risk** — Potential event with negative impact on security

## ⭐ S

- **SAST (Static Application Security Testing)** — Analyzing source code for vulnerabilities
- **Secure software development lifecycle (SSDLC)** — Integrating security into SDLC
- **Security control** — Measure to reduce security risk
- **Security testing** — Testing to determine if system protects data and functionality
- **Social engineering** — Manipulating people to reveal confidential information
- **SQL injection** — Injection of SQL commands through input data

## ⭐ T

- **Threat** — Potential cause of unwanted incident
- **Threat actor** — Entity that can act on a threat

## ⭐ V

- **Vulnerability** — Weakness that can be exploited by a threat
- **Vulnerability scanning** — Automated process to identify known vulnerabilities

## ⭐ W

- **White-box testing** — Testing with knowledge of internal structure
