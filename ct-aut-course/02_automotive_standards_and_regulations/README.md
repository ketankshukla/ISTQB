# Chapter 2 — Automotive Standards and Regulations

## Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2, K3
- **Focus:** ISO 26262, ASPICE, AUTOSAR, ASIL levels, process assessment

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Explain ISO 26262 and ASIL levels
2. Describe ASPICE process assessment model
3. Understand AUTOSAR architecture
4. Apply ASIL to test planning
5. Map standards to testing activities
6. Explain regulatory requirements for automotive software

---

## Key Concepts

- ISO 26262 functional safety
- ASIL A-D
- ASPICE levels
- AUTOSAR architecture
- Safety goals and requirements
- Process assessment
