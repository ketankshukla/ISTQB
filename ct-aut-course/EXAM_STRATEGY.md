# CT-AuT Exam Strategy

## Exam Format

- 40 questions in 60 minutes
- Closed book
- Single best answer (some "choose N")
- 65% pass mark (26/40 correct)

## Time Budget

- 60 minutes / 40 questions = 1.5 minutes per question
- Flag uncertain questions, return at the end
- Reserve 5 minutes for review

## K-Level Patterns

- **K1 (Remember):** ISO 26262, ASPICE, AUTOSAR terms and levels
- **K2 (Understand):** Why traceability is needed; how ASIL affects testing
- **K3 (Apply):** Select test techniques for an ASIL level; design test environment

## Common Distractor Patterns

- **"ISO 26262 is only for hardware"** — False; it covers software and system
- **"Higher ASIL = less testing"** — False; higher ASIL = more rigorous testing
- **"ASPICE is a test tool"** — False; it's a process assessment model

## Pre-Exam Checklist

- [ ] All mock exams completed with 80%+
- [ ] Glossary terms memorized
- [ ] Exam traps reviewed
- [ ] Case studies completed

## Exam Day Advice

- Read each question fully before answering
- Watch for absolutes ("always," "never") — they are usually wrong
- For "choose N" questions, verify you selected exactly N
