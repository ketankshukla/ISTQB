# CTFL-PT Exam Strategy

## Exam Format

- 20 questions in 40 minutes
- Closed book
- Single best answer
- 65% pass mark (13/20 correct)

## Time Budget

- 40 minutes / 20 questions = 2 minutes per question
- Flag uncertain questions, return at the end
- Reserve 5 minutes for review

## K-Level Patterns

- **K1 (Remember):** Performance test types, load terminology, monitoring metrics
- **K2 (Understand):** Why different test types are used; how metrics indicate problems
- **K3 (Apply):** Select appropriate test type for a scenario; calculate capacity

## Common Distractor Patterns

- **"Performance testing is only load testing"** — False; includes stress, spike, endurance, scalability, volume
- **"Throughput and response time are the same"** — False; throughput = requests/time, response time = duration/request
- **"A single tool handles everything"** — False; need load generator, monitor, analyzer

## Pre-Exam Checklist

- [ ] All mock exams completed with 80%+
- [ ] Glossary terms memorized
- [ ] Exam traps reviewed
- [ ] Case studies completed

## Exam Day Advice

- Read each question fully before answering
- Watch for absolutes ("always," "never")
- For calculation questions, estimate first
