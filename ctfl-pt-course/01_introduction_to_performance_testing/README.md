# 🌟 Chapter 1 — Introduction to Performance Testing

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~3 questions out of 20)
- **K-levels:** K1, K2
- **Focus:** What is performance testing, why it matters, key concepts

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Define performance testing
2. Explain why performance testing is important
3. Identify key performance metrics
4. Describe factors that affect performance
5. Recognize when performance testing is needed

---

## ⭐ Key Concepts

- Performance testing definition
- Performance metrics (response time, throughput, resource utilization)
- Factors affecting performance (hardware, software, network, data volume)
- Business impact of poor performance
- Types of performance failures
