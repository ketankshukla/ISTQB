# Chapter 2 — Performance Test Types and Objectives

## Exam Weighting

- **Estimated weight:** ~20% (~4 questions out of 20)
- **K-levels:** K2, K3
- **Focus:** Load, stress, spike, endurance, scalability, volume testing

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Describe different performance test types
2. Explain objectives of each test type
3. Select appropriate test type for a scenario
4. Identify when each type is used
5. Understand workload models

- Load testing
- Stress testing
- Spike testing
- Endurance testing
- Scalability testing
