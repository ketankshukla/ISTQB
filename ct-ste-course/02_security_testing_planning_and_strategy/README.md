# Chapter 2 — Security Testing Planning and Strategy

## Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2, K3
- **Focus:** Risk assessment, test planning, scope, strategy, compliance

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Perform security risk assessments
2. Define security test scope based on risk
3. Develop a security test strategy
4. Align security testing with business risk
5. Plan security testing for compliance
6. Select appropriate security testing approaches

---

## Key Concepts

- Risk assessment methodology
- Security test scope
- Test strategy components
- Compliance-driven testing
- White-box vs black-box vs grey-box
- Security test levels
