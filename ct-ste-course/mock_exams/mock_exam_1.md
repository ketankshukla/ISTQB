# CT-STE Mock Exam 1

**Instructions:** 40 questions, 60 minutes, single best answer.

---

### Q1-Q10

1. Security testing: A) Only functional B) Vulnerabilities + controls C) Only performance D) Only usability — **Vulnerabilities + controls**
2. CIA triad missing: A) Confidentiality B) Integrity C) Availability D) Scalability — **Scalability**
3. Confidentiality: A) Integrity B) Access control C) Availability D) Accountability — **Access control**
4. Nation-state: A) Low capability B) High capability, espionage C) Only fun D) Only accidental — **High capability, espionage**
5. Penetration: A) Only form B) One type of security testing C) Only networks D) Only compliance — **One type of security testing**
6. Integrity: A) Access B) No improper alteration C) Availability D) Authentication — **No improper alteration**
7. Insider: A) External B) Legitimate user abuse C) Only malware D) Only nation-state — **Legitimate user abuse**
8. Injection: A) Weak encryption B) Input as command C) Only logging D) Only sessions — **Input as command**
9. Security vs functional: A) Same cases B) Malicious inputs C) Only after release D) Only manual — **Malicious inputs**
10. Non-repudiation: A) Encryption B) Provable actions C) Availability D) Admin access — **Provable actions**

13. White-box: A) External attacker B) Source code review C) Only production D) Only compliance — **Source code review**

17. Threat modeling: A) Only before release B) Design phase C) Only after coding D) Only maintenance — **Design phase**
18. Third-party libs: A) Trusted B) Scan for vulns C) Only manual D) Only production — **Scan for vulns**
