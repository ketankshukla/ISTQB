# Chapter 6 — Security Testing in the SDLC

## Learning Objectives

By the end of this chapter, you should be able to:

2. Apply secure development lifecycle (SSDLC) principles
3. Create security requirements
4. Perform threat modeling
5. Apply security testing in DevOps
6. Manage security technical debt
