# Chapter 1 — Introduction to Security Test Engineering

## Exam Weighting

- **Estimated weight:** ~10% (~4 questions out of 40)
- **K-levels:** K1, K2
- **Focus:** Security fundamentals, CIA triad, threat actors, security testing scope

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Define security test engineering
2. Explain the CIA triad
3. Identify types of threat actors
4. Describe the scope of security testing
5. Distinguish security testing from functional testing

## Key Concepts

- CIA triad (Confidentiality, Integrity, Availability)
- Threat actors and motivations
- Security testing vs penetration testing
- Security testing objectives
- OWASP Top 10 overview
