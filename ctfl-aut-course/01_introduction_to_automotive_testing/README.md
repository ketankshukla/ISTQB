# 🌟 Chapter 1 — Introduction to Automotive Testing

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~3 questions out of 20)
- **K-levels:** K1, K2
- **Focus:** What is automotive testing, key concepts, differences from general testing

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Describe the automotive software landscape
2. Explain why automotive testing is different from general software testing
3. Identify key automotive components (ECUs, vehicle networks)
4. Recognize safety-critical characteristics
5. Understand the role of standards in automotive testing

---

## ⭐ Key Concepts

- Automotive software characteristics
- ECUs and vehicle networks
- Safety-critical software
- Differences between automotive and general testing
- Overview of standards (ISO 26262, ASPICE)
