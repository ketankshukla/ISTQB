# Chapter 2 — Automotive Software Basics

## Exam Weighting

- **Estimated weight:** ~20% (~4 questions out of 20)
- **K-levels:** K2, K3
- **Focus:** ECUs, vehicle networks, safety concepts, ASIL

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Describe ECU types and functions
2. Explain vehicle network types
3. Understand safety integrity levels (ASIL)
4. Identify safety-critical vs non-safety-critical components
5. Describe the V-model in automotive context

---

## Key Concepts

- ECU types and architecture
- CAN, LIN, FlexRay, Ethernet
- ASIL levels (A to D)
- Safety-critical vs QM
- V-model for automotive
