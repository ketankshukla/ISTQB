# ♟️ CTFL-AuT Exam Strategy

## 🎓 Exam Format

- 20 questions in 40 minutes
- Closed book
- Single best answer
- 65% pass mark (13/20 correct)

## ⭐ Time Budget

- 40 minutes / 20 questions = 2 minutes per question
- Flag uncertain questions, return at the end
- Reserve 5 minutes for review

## 🪜 K-Level Patterns

- **K1 (Remember):** ECU types, vehicle networks, safety levels, standards
- **K2 (Understand):** How automotive testing differs from general testing
- **K3 (Apply):** Select appropriate test level for an automotive component

## ⭐ Common Distractor Patterns

- **"Automotive testing is the same as general software testing"** — False; safety-critical, standards-driven, hardware-software integration
- **"Unit testing is not needed for automotive"** — False; required for all safety-critical functions
- **"Only HIL is used for automotive testing"** — False; MIL, SIL, PIL, and vehicle testing also used

## 🎓 Pre-Exam Checklist

- [ ] All mock exams completed with 80%+

## 🎓 Exam Day Advice

- Read each question fully before answering
- Watch for absolutes ("always," "never")
- Remember: automotive testing has unique safety and regulatory requirements
