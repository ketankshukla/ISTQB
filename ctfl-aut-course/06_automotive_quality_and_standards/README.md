# Chapter 6 — Automotive Quality and Standards

## Exam Weighting

- **Estimated weight:** ~15% (~3 questions out of 20)
- **K-levels:** K2, K3
- **Focus:** ISO 26262, ASPICE, AUTOSAR, quality metrics

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Describe ISO 26262 and its relationship to testing
2. Explain ASPICE and process assessment
3. Understand AUTOSAR architecture
4. Identify automotive quality metrics
5. Describe audit and assessment preparation

---

## Key Concepts

- ISO 26262 functional safety
- ASPICE process model
- AUTOSAR architecture
- Quality metrics
- Audit readiness
