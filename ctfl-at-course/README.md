# ISTQB CTFL-AT v4.0 — Agile Tester Self-Study Course

Complete self-study course for the **ISTQB Certified Tester Foundation Level — Agile Tester (CTFL-AT)** examination.

---

## Exam Facts

| Parameter | Value |
|-----------|-------|
| Full name | ISTQB Certified Tester Foundation Level — Agile Tester |
| Syllabus version | v4.0 |
| Total questions | 40 |
| Duration | 60 minutes |
| Pass mark | 65% (26 points) |
| Question types | Multiple choice (single best answer, scenario-based) |
| Materials allowed | None (closed book) |
| Negative marking | No |
| Prerequisites | ISTQB CTFL v4.0 (or compatible Foundation Level) |

---

## Course Structure

This course is organized into 3 chapters aligned with the CTFL-AT v4.0 syllabus:

| Chapter | Topic | Weight | K-Levels |
|---------|-------|--------|----------|
| 1 | Agile Testing Fundamentals | ~35% | K2, K3 |
| 2 | Agile Testing Methods and Techniques | ~45% | K2, K3 |
| 3 | Agile Testing Tools and Automation | ~20% | K2, K3 |

**Key insight:** Chapter 2 (Methods and Techniques) is the highest-weighted chapter. Master ATDD/BDD, exploratory testing, and collaboration practices for a strong floor.

---

## How to Use This Course

1. Follow the `STUDY_PLAN.md` — designed for 2-3 weeks of study at 8-12 hours/week.
2. Read each chapter's `lessons.md` thoroughly — primary learning material.
3. Review the chapter `README.md` for exam weighting and learning objectives.
4. Study `exam_traps.md` to avoid common mistakes.
5. Complete all `practice_questions.md` — time yourself.
6. Work through all 3 case studies — they integrate multiple chapters.
7. Attempt the mock exams under timed conditions.
8. Review `EXAM_STRATEGY.md` in the final week.

## Certification Context

The CTFL-AT is a Foundation Level extension certification. After achieving CTFL-AT, you may consider:

- **CTAL-TA** (Advanced Test Analyst) — for advanced test design and analysis
- **CTAL-AT** (Agile Technical Tester) — for technical testing in agile
- **CT-TAE** (Test Automation Engineering) — for automation specialization
- **CT-GenAI** (AI in Testing) — for using GenAI in testing

For a complete certification path map, see `ISTQB_Certification_Paths.md` in the repository root.
