# Chapter 6 — Acceptance Testing Tools and Automation

## Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2 (understand)
- **Focus:** BDD tools, test management, traceability, virtualization, collaboration tools

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Identify tools that support acceptance testing
2. Describe BDD tools (Cucumber, SpecFlow)
3. Explain test management and traceability tools
4. Describe test environment virtualization and data management
5. Identify collaboration tools for acceptance testing
6. Select appropriate tools for different scenarios

---

## Key Concepts

- BDD tools (Cucumber, SpecFlow, Behave, JBehave)
- Test management tools (Jira, TestRail, Zephyr, Xray)
- Requirement management tools (DOORS, Jama, ReqView)
- Traceability matrix tools
- Test data management and virtualization
- Collaboration and communication tools
- CI/CD integration for acceptance tests
