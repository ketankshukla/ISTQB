# Chapter 4 — Contract Acceptance Testing (CAT)

## Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** SLAs, contractual criteria, supplier testing, acceptance in outsourcing

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Describe Contract Acceptance Testing (CAT) objectives
2. Identify contractual acceptance criteria
3. Explain SLAs and their role in acceptance
4. Describe acceptance testing in outsourced/offshore development
5. Apply CAT techniques to scenarios

---

## Key Concepts

- Contract Acceptance Testing (CAT)
- Service Level Agreements (SLAs)
- Key Performance Indicators (KPIs)
- Contractual acceptance criteria
- Supplier and customer responsibilities
- Outsourcing and offshoring acceptance
- Penalties and incentives
