# 🌟 Chapter 1 — Introduction to Acceptance Testing

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K1 (remember), K2 (understand)
- **Focus:** Acceptance testing fundamentals, objectives, types, scope

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Define acceptance testing and its objectives
2. Differentiate acceptance testing from other test levels
3. Identify types of acceptance testing (alpha, beta, UAT, OAT)
4. Describe acceptance criteria and their importance
5. Identify stakeholders in acceptance testing

---

## ⭐ Key Concepts

- Acceptance testing definition and objectives
- Alpha and beta testing
- User Acceptance Testing (UAT)
- Operational Acceptance Testing (OAT)
- Acceptance criteria
- Stakeholders and business representatives
- Acceptance vs. system vs. integration testing
