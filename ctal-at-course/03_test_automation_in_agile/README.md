# Chapter 3 — Test Automation in Agile

## Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Test automation strategy, frameworks, maintainability, test data management, automation in agile teams

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Explain test automation strategy in agile
2. Describe automation framework types (data-driven, keyword-driven, hybrid)
3. Explain maintainability practices (Page Object Model, abstraction layers)
4. Describe test data management in agile automation
5. Explain the role of automation in regression testing and continuous feedback

---

## Key Concepts

- Test pyramid and automation strategy
- Framework types: linear, modular, data-driven, keyword-driven, hybrid
- Maintainability: POM, abstraction, separation of concerns
- Test data: generation, management, isolation, cleanup
- Regression automation and smoke tests
- Automation metrics and ROI
