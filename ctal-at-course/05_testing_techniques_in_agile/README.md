# 🎯 Chapter 5 — Testing Techniques in Agile

## 🎓 Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Exploratory testing, session-based testing, pairwise testing, non-functional testing in agile, testing quadrants

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Explain exploratory testing and its role in agile
2. Describe session-based test management (SBTM)
3. Explain pairwise and combinatorial testing in agile
4. Describe non-functional testing in agile (performance, security)
5. Explain the agile testing quadrants

---

## ⭐ Key Concepts

- Exploratory testing: simultaneous test design and execution
- SBTM: charters, time boxes, debriefs
- Pairwise testing: reducing combinations
- Testing quadrants: business/tech x support/critique
- Non-functional testing in agile: performance, security, usability
