# 📱 Chapter 3 — Mobile Functional Testing

## 🎓 Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2, K3
- **Focus:** UI testing, navigation, input methods, compatibility, installation

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Test mobile UI components and layouts
2. Validate touch input and gestures
3. Test navigation and orientation changes
4. Verify compatibility across devices and OS versions
5. Test installation, update, and uninstallation
6. Validate push notifications and deep links

---

## ⭐ Key Concepts

- Touch targets and accessibility
- Gesture recognition
- Screen orientation and responsive design
- Device compatibility matrix
- App store installation testing
- Push notification testing
- Deep linking and universal links
