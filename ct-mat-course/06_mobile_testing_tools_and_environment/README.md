# Chapter 6 — Mobile Testing Tools and Environment

## Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K1, K2
- **Focus:** Dev tools, device labs, debugging, beta distribution, test management

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Identify mobile development and testing tools
2. Describe device farm and cloud testing services
3. Use debugging and profiling tools
4. Explain beta testing distribution
5. Select appropriate tools for mobile testing scenarios

---

## Key Concepts

- Android Studio and Xcode tools
- ADB (Android Debug Bridge)
- Device farms (Firebase, AWS, BrowserStack, Sauce Labs)
- Profiling and debugging tools
- Beta testing platforms (TestFlight, Google Play Console)
- Test management for mobile
