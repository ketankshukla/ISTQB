# Chapter 4 — Mobile Non-Functional Testing

## Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2, K3
- **Focus:** Performance, battery, security, usability, accessibility, network

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Test mobile app performance (startup, memory, CPU)
2. Measure and optimize battery consumption
3. Test network behavior and offline functionality
4. Validate mobile security (data storage, communication, authentication)
5. Test usability and accessibility
6. Test under various environmental conditions

---

## Key Concepts

- App startup time and cold/warm/hot start
- Memory usage and leaks
- Battery drain testing
- Network performance and offline mode
- Mobile security (OWASP MASVS)
- Accessibility (VoiceOver, TalkBack)
