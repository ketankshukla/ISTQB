# 📱 Chapter 2 — Mobile Platforms and Architecture

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K1, K2
- **Focus:** iOS and Android architecture, app components, lifecycle, permissions

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Describe iOS and Android architecture differences
2. Identify mobile app components (activities, fragments, view controllers)
3. Explain the mobile app lifecycle
4. Describe permission models on iOS and Android
5. Identify platform-specific testing considerations

---

## ⭐ Key Concepts

- iOS architecture (UIKit, SwiftUI, AppDelegate, SceneDelegate)
- Android architecture (Activities, Fragments, Services, Intents)
- Mobile app lifecycle states
- Permission models (iOS permissions, Android runtime permissions)
- Platform-specific UI guidelines (Human Interface Guidelines, Material Design)
