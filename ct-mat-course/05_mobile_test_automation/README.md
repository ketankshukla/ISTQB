# Chapter 5 — Mobile Test Automation

## Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2, K3
- **Focus:** Automation frameworks, Page Object Model, cross-platform, CI/CD integration

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Select appropriate mobile automation frameworks
2. Implement Page Object Model for mobile
3. Design cross-platform test strategies
4. Integrate mobile tests into CI/CD
5. Handle flaky tests in mobile automation
6. Manage test devices and emulators/simulators

---

## Key Concepts

- UI automation frameworks (Appium, Espresso, XCUITest, Detox)
- Page Object Model for mobile
- Cross-platform automation trade-offs
- Emulator vs simulator vs real device
- CI/CD integration for mobile
- Flaky test management
