3. Device fragmentation worst on: A) iOS B) Android C) Windows D) BlackBerry — **Android**
4. Hybrid app: A) One platform B) Web app in native container C) Only browser D) Compiled native — **Web app in native container**
5. Unique mobile challenge: A) Database normalization B) Incoming calls/notifications C) Server config D) Cable testing — **Incoming calls/notifications**

8. TestFlight: A) Android beta B) iOS beta C) Web testing D) Performance — **iOS beta**
9. Cross-platform native compiles to: A) Web HTML B) Native code C) Java bytecode D) Python — **Native code**
