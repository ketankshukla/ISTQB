# Chapter 4 — Testing Financial Calculations and Algorithms

## Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2, K3
- **Focus:** Interest, amortization, VaR, derivatives Greeks, FX, precision, rounding

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Test interest and amortization calculations
2. Verify derivative pricing and Greeks calculations
3. Test FX conversions and cross rates
4. Validate risk metrics (VaR, exposure)
5. Identify precision and rounding issues
6. Design test data for financial calculations

---

## Key Concepts

- Simple and compound interest
- Loan amortization schedules
- Bond pricing and accrued interest
- Derivatives Greeks (delta, gamma, theta, vega)
- Value-at-Risk (VaR) methods
- FX calculations and cross rates
- Decimal precision and rounding
- Boundary value analysis for financial data
