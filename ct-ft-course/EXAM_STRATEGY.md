# ♟️ CT-FT Exam Strategy

## 🎓 Exam Format

- 40 questions in 60 minutes
- Closed book
- Single best answer (some "choose N")
- 65% pass mark (26/40 correct)

## ⭐ Time Budget

- 60 minutes / 40 questions = 1.5 minutes per question
- Flag uncertain questions, return at the end
- Reserve 5 minutes for review

## 🪜 K-Level Patterns

- **K1 (Remember):** Definitions of financial terms, regulations, acronyms
- **K2 (Understand):** How compliance affects testing, why precision matters, when to use specific techniques
- **K3 (Apply):** Calculate rounding errors, select test data for financial scenarios, determine regulatory test coverage

## ⭐ Common Distractor Patterns

- **"Accuracy" vs "Precision"** — The exam tests whether you know precision is about decimal places, accuracy is about correctness
- **"Functional" vs "Regulatory"** — Tests that verify SOX compliance are regulatory, not purely functional
- **"Unit" vs "Integration"** — Interest calculation at unit level vs end-to-end loan processing

## 🎓 Pre-Exam Checklist

- [ ] All mock exams completed with 80%+
- [ ] Glossary terms memorized
- [ ] Exam traps reviewed
- [ ] Case studies completed

## 🎓 Exam Day Advice

- Read each question fully before answering
- Watch for absolutes ("always," "never") — they are usually wrong
- For "choose N" questions, verify you selected exactly N
