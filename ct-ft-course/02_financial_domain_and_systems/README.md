# Chapter 2 — Financial Domain and Systems

## Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K1, K2
- **Focus:** Banking, trading, payment, insurance systems; financial instruments; market participants

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Describe core banking system components
2. Identify types of financial instruments
3. Explain trading system architecture (front/middle/back office)
4. Describe payment system types and flows
5. Identify key domain terminology

---

## Key Concepts

- Core banking systems
- Financial instruments (equities, bonds, derivatives, forex)
- Trading system layers
- Payment systems and settlement cycles
- Insurance systems
- Financial market participants
