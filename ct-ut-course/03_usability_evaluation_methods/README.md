# Chapter 3 — Usability Evaluation Methods

## Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** User-based methods, expert-based methods, comparison of methods

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Describe user-based evaluation methods
2. Describe expert-based evaluation methods
3. Explain the strengths and weaknesses of each method
4. Select the appropriate method for a given situation
5. Explain the concept of pluralistic walkthrough

---

## Key Concepts

- Think-aloud, co-discovery, retrospective testing
- Heuristic evaluation, cognitive walkthrough, guideline review
- Moderated vs. unmoderated testing
- Cost-benefit comparison of methods
- Pluralistic walkthrough
