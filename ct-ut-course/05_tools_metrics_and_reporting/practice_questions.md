## Answers and Rationales

### Q1
**Answer: B**

**Rationale:** SUS score ranges from 0 to 100.

### Q2
**Answer: B**

**Rationale:** Industry average SUS score is approximately 68.

### Q3
**Answer: B**

**Rationale:** NPS = % Promoters - % Detractors.

### Q4
**Answer: B**

**Rationale:** Task success rate = (Successful completions / Attempts) x 100%.

### Q5
**Answer: B**

**Rationale:** Slip = accidental error; knew right action but did wrong one.

### Q6
**Answer: B**

**Rationale:** (60+75+70+80+65) / 5 = 350 / 5 = 70.

### Q7
**Answer: B**

**Rationale:** Time on task only meaningful with success rate (fast failure is not good).

### Q8
**Answer: B**

**Rationale:** Quantitative = identifies what happened, enables comparison.

### Q9
**Answer: B**

**Rationale:** CSAT = customer satisfaction with a specific experience.

### Q10
**Answer: B**

**Rationale:** Session recording tools = understand real behavior at scale.

### Q11
**Answer: B**

**Rationale:** Developer reports = specific UI elements, reproduction steps, recordings.

### Q12
**Answer: B**

**Rationale:** Mistake = conscious error based on wrong understanding.

### Q13
**Answer: C**

**Rationale:** CSUQ provides diagnostic info on usefulness, information quality, interface quality.

### Q14
**Answer: B**

**Rationale:** Report must include both metrics and qualitative emotional feedback.

### Q15
**Answer: B**

**Rationale:** Remote unmoderated platforms = quick feedback from many participants.

### Q16
**Answer: B**

**Rationale:** 78% = industry average for web applications.

### Q17
**Answer: C**

**Rationale:** 9-10 = Promoters.

### Q18
**Answer: B**

**Rationale:** Executive summary = brief overview for stakeholders.

### Q19
**Answer: B**

**Rationale:** Wrong understanding of what "Save" does = mistake.

### Q20
**Answer: B**

**Rationale:** Quantitative identifies problems; qualitative explains why and guides solutions.
