# Chapter 5 — Tools, Metrics and Reporting

## Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Questionnaires, task metrics, error metrics, reporting structure, tools

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Describe common usability questionnaires (SUS, NPS, CSAT)
2. Explain task-based metrics (success rate, time on task, error rate)
3. Describe error classification and analysis
4. Explain the structure and content of usability test reports
5. Identify tools for usability testing

---

## Key Concepts

- SUS, NPS, CSAT, CSUQ
- Task success rate, time on task, error rate
- Error classification (slips vs. mistakes)
- Quantitative vs. qualitative data
- Usability testing tools (remote, lab, analytics)
