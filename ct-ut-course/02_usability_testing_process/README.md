# Chapter 2 — Usability Testing Process

## Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Planning, preparation, execution, analysis, reporting

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Describe the phases of the usability testing process
2. Explain how to plan a usability test
3. Describe participant recruitment and ethical considerations
4. Explain test execution procedures
5. Describe analysis and reporting of usability test results

---

## Key Concepts

- Formative vs. summative testing
- Planning: objectives, scope, method selection
- Preparation: test environment, scenarios, pilot testing
- Execution: moderation techniques, data collection
- Analysis: severity ratings, prioritization
- Reporting: structure, audience-appropriate communication
