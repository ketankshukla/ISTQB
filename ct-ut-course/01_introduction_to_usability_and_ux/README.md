# 🌟 Chapter 1 — Introduction to Usability and User Experience

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand)
- **Focus:** Definitions, principles, ISO standards, UX vs. usability

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Define usability and user experience
2. Explain the ISO 9241-11 definition of usability
3. Describe the ISO 25010 quality characteristics
4. Explain the relationship between usability and user experience
5. Identify key usability principles and heuristics

---

## ⭐ Key Concepts

- Usability vs. user experience (UX)
- ISO 9241-11: effectiveness, efficiency, satisfaction
- ISO 25010 quality model
- Nielsen's 10 heuristics
- UX Honeycomb model
- Cognitive load, mental models, affordances
