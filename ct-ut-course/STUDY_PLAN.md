## 📚 Study Techniques That Work

**1. Active Recall**
After reading a section, close the file and write down what you remember. Check against the text.

**2. Spaced Repetition**
Re-visit practice questions from previous chapters every 3-4 days.

**3. Interleaving**
Mix topics during study sessions. Do some usability principles questions, then some accessibility questions.

**4. Teach-Back**
After each chapter, explain the key concepts aloud as if teaching a colleague.

**5. Connect to Real Examples**
Think about applications you use and how the usability concepts apply. Note good and bad examples.

## 🎓 Exam Day Checklist

- [ ] 7+ hours of sleep the night before
- [ ] Review `exam_traps.md` one final time
- [ ] Arrive 15 minutes early (or set up exam space)
- [ ] Read each question twice before selecting an answer
- [ ] Flag uncertain questions and return to them
- [ ] Budget ~1.5 minutes per question (leaves 10 minutes for review)
- [ ] Trust your first instinct unless you find clear evidence to change
