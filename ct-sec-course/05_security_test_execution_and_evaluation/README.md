# 🔒 Chapter 5 — Security Test Execution and Evaluation

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** CVSS, vulnerability reporting, remediation, verification testing

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Describe CVSS scoring components
2. Evaluate and prioritize vulnerabilities
3. Create effective security test reports
4. Plan remediation activities
5. Perform verification testing after fixes
6. Describe false positives and false negatives in security testing

---

## ⭐ Key Concepts

- CVSS v3.1 base, temporal, environmental metrics
- Vulnerability prioritization
- Security test reporting
- Remediation planning
- Verification testing
- False positives vs. false negatives
