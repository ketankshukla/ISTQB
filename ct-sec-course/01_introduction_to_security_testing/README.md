# 🌟 Chapter 1 — Introduction to Security Testing

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K1 (remember), K2 (understand)
- **Focus:** Security fundamentals, CIA triad, security testing vs. functional testing, OWASP basics

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Explain the CIA triad (Confidentiality, Integrity, Availability)
2. Describe the relationship between security and functionality
3. Identify common security terminology
4. Recognize the OWASP Top 10 categories
5. Describe why security testing is necessary

---

## ⭐ Key Concepts

- Confidentiality, Integrity, Availability (CIA)
- Authentication, Authorization, Non-repudiation
- Vulnerability, Threat, Risk, Exploit, Attack
- Security testing objectives
- OWASP Top 10 overview
- Security testing vs. functional testing
