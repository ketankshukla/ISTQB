# 🛠️ Chapter 6 — Security Testing Tools and Automation

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand)
- **Focus:** SAST/DAST/IAST/SCA tools, web app scanners, CI/CD security, fuzzing tools

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Identify categories of security testing tools
2. Describe SAST, DAST, IAST, and SCA tools
3. Explain web application security scanners
4. Describe CI/CD security integration
5. Recognize fuzzing and network security tools
6. Select appropriate tools for different scenarios

---

## ⭐ Key Concepts

- SAST tools (SonarQube, Checkmarx, Fortify)
- DAST tools (OWASP ZAP, Burp Suite)
- IAST tools (Contrast, Seeker)
- SCA tools (Snyk, OWASP Dependency-Check)
- Web application scanners
- CI/CD security automation
- Fuzzing tools (AFL, libFuzzer)
- Network scanners (Nmap, Nessus)
