# 🧪 Chapter 4 — Gambling Testing Types and Tools

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand)
- **Focus:** Functional and non-functional testing in gambling, game-specific test types, testing tools, automation

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Describe functional testing specific to gambling products
2. Describe non-functional testing in a gambling context
3. Explain the role of game-specific testing (RNG testing, game math verification)
4. Describe gambling testing tools and their uses
5. Explain test automation considerations for gambling products

---

## ⭐ Key Concepts

- Functional testing: game mechanics, payouts, bonus features, progressive jackpots
- Non-functional testing: performance under load, security, usability, compatibility
- Game-specific testing: RNG testing, reel strip verification, RTP verification, odds verification
- Tools: PAR sheet analyzers, RNG test suites, platform test automation frameworks
- Automation: test data generation, regression testing, CI/CD in gambling
