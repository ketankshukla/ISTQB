# Chapter 2 — Game Mathematics and RNG Testing

## Exam Weighting

- **Estimated weight:** ~25% (~10 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Probability, odds, RTP calculation, RNG testing, game math verification

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Calculate basic probabilities for gambling outcomes
2. Explain how RTP is calculated from a paytable
3. Describe the properties of a suitable RNG
4. Describe RNG testing methods (statistical tests, uniformity, unpredictability)
5. Explain how game mathematics is verified by testing
6. Calculate expected value and house edge from game rules

---

## Key Concepts

- Probability, odds, and expected value
- RTP calculation from paytables
- House edge calculation
- RNG properties: randomness, uniformity, unpredictability
- RNG testing: chi-square, diehard tests, uniformity testing
- Game math verification: PAR sheets, reel strips, symbol frequency
- Volatility and its relationship to game mathematics
