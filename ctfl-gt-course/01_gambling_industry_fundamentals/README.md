# 🏗️ Chapter 1 — Gambling Industry Fundamentals

## 🎓 Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K1 (remember), K2 (understand)
- **Focus:** Gambling domains, game types, key terminology, industry structure

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Describe the main gambling domains (casino, sports betting, lottery, poker, bingo)
2. Describe different game types within each domain
3. Define key gambling industry terms (RTP, house edge, volatility, hit frequency)
4. Describe the structure of the gambling industry (operators, suppliers, regulators, testers)
5. Explain the role of testing in the gambling industry

---

## ⭐ Key Concepts

- Gambling domains: casino, sports betting, lottery, poker, bingo, fantasy sports
- Game types: slots, table games, live dealer, instant win, virtual sports
- Key terms: RTP, house edge, volatility, hit frequency, paylines, paytables
- Industry stakeholders: operators, platform providers, game studios, regulators, testing labs
- Testing objectives: fairness, compliance, player protection, fraud prevention
