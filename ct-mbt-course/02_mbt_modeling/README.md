# 📐 Chapter 2 — MBT Modeling

## 🎓 Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Model types, notations, quality attributes, abstraction

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Describe different types of test models
2. Explain common modeling notations
3. Identify model quality attributes
4. Choose appropriate model types for different test objectives
5. Understand abstraction levels in modeling

---

## ⭐ Key Concepts

- State transition models, decision tables, process models, feature models
- UML state machines, activity diagrams, BPMN
- Model quality: correctness, representativeness, interpretability, maintainability
- Control flow vs. data flow models
- Test-relevant aspects and abstraction
