# 🚀 Chapter 4 — Test Implementation and Execution

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Abstract vs. concrete tests, concretization, execution, failure analysis

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Explain abstract and concrete test cases
2. Describe the concretization process
3. Explain test execution approaches
4. Understand the oracle problem in MBT
5. Analyze test failures

---

## ⭐ Key Concepts

- Abstract test cases vs. concrete test cases
- Concretization (data, mapping)
- Manual, automated, and simulator-based execution
- Model as oracle
- Failure analysis: SUT bug, model bug, concretization error
