# 🌟 Chapter 1 — Introduction to Model-Based Testing

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand)
- **Focus:** Definitions, benefits, limitations, MBT in the test process

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Define model-based testing and related terms
2. Explain the benefits and limitations of MBT
3. Describe how MBT fits into the test process
4. Differentiate between online and offline MBT
5. Understand the relationship between models and the SUT

---

## ⭐ Key Concepts

- MBT definition and scope
- Benefits: automation, early defect detection, coverage, maintainability
- Limitations: model creation effort, expertise, not universal
- Online vs. offline MBT
- Test process integration
- Model vs. SUT relationship
