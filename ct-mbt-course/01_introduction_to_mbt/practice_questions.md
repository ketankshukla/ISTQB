### 🔷 Q20 (MBT-1.4, K3)

A game development team wants to test AI opponent behavior that adapts to player actions. The BEST MBT approach is:

A. Offline MBT with a fixed set of test cases
B. Online MBT that adapts tests based on AI responses
C. No MBT because games are too complex
D. Manual testing only

---

## ✔️ Answers and Rationales

### 🔷 Q1
**Answer: B**

**Rationale:** MBT = test cases derived from a model describing aspects of the SUT.

### 🔷 Q2
**Answer: B**

**Rationale:** Primary benefit = automation of test design, reducing manual effort.

### 🔷 Q3
**Answer: B**

**Rationale:** Modeling forces requirement clarification, revealing ambiguities early.

### 🔷 Q4
**Answer: B**

**Rationale:** Offline = generated before execution, can be stored and reviewed.

### 🔷 Q5
**Answer: B**

**Rationale:** Online = non-deterministic systems where adaptive testing is needed.

### 🔷 Q6
**Answer: B**

**Rationale:** Limitation = model creation requires time, skill, and domain knowledge.

### 🔷 Q7
**Answer: B**

**Rationale:** MBT primarily supports test analysis and test design.

### 🔷 Q8
**Answer: B**

**Rationale:** Regulatory documentation = offline MBT (reviewable, storable test cases).

### 🔷 Q9
**Answer: B**

**Rationale:** MBT advantage = update model once, regenerate all tests.

### 🔷 Q10
**Answer: B**

**Rationale:** Model = represents SUT aspects relevant for testing at appropriate abstraction.

### 🔷 Q11
**Answer: C**

**Rationale:** Simple static webpage = not stateful, not rule-based; MBT effort exceeds benefit.

### 🔷 Q12
**Answer: A**

**Rationale:** Model-based coverage (states, transitions) is measurable.

### 🔷 Q13
**Answer: C**

**Rationale:** Online MBT adapts based on SUT responses during execution.

### 🔷 Q14
**Answer: B**

**Rationale:** Model serves as oracle for predicting expected results.

### 🔷 Q15
**Answer: B**

**Rationale:** Test quality depends on model quality and completeness.

### 🔷 Q16
**Answer: B**

**Rationale:** Partial models are valid starting points for iterative improvement.

### 🔷 Q17
**Answer: B**

**Rationale:** MBT + risk-based testing = focus modeling on high-risk areas.

### 🔷 Q18
**Answer: B**

**Rationale:** MBT most relevant during test analysis and test design.

### 🔷 Q19
**Answer: B**

**Rationale:** No expertise = start with pilot, training, and simpler notations.

### 🔷 Q20
**Answer: B**

**Rationale:** Adaptive AI behavior = online MBT that adapts based on responses.
