# ✅ Chapter 3 — Test Selection Criteria

## 🎓 Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Coverage criteria, transition-based, path-based, data-driven selection

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Explain different test selection criteria
2. Apply coverage criteria to models
3. Describe data-driven selection techniques
4. Understand the trade-off between coverage and test case count
5. Explain random vs. systematic selection

## ⭐ Key Concepts

- State coverage, transition coverage, path coverage
- Boundary value testing on models
- Equivalence partitioning for model data
- Transition pair coverage
- Random vs. systematic testing
- Test case length
