# ✅ Chapter 3 — Selection Criteria for Test Case Generation

## 🎓 Exam Weighting

- **Estimated weight:** ~25% (~10 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Coverage criteria for state machines, transition coverage, transition pair coverage, boundary testing on models

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Describe state transition coverage criteria (0-switch, 1-switch, 2-switch)
2. Describe transition pair coverage
3. Describe boundary testing applied to models
4. Calculate coverage percentages for generated test suites
5. Explain the relationship between coverage criteria and test suite size

---

## ⭐ Key Concepts

- 0-switch, 1-switch, 2-switch coverage
- Transition pair coverage
- Full state coverage vs. full transition coverage
- Boundary testing on model variables
- Coverage calculation and reporting
- Trade-offs between coverage and test suite size
