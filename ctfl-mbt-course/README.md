# 📚 ISTQB CTFL-MBT — Model-Based Tester Self-Study Course

Complete self-study course for the **ISTQB Certified Tester Foundation Level — Model-Based Tester (CTFL-MBT)** examination.

---

## 🎓 Exam Facts

| Parameter | Value |
|-----------|-------|
| Full name | ISTQB Certified Tester Foundation Level — Model-Based Tester |
| Syllabus version | v1.0 |
| Total questions | 40 |
| Duration | 60 minutes |
| Pass mark | 65% (26 points) |
| Question types | Multiple choice (single best answer) |
| Materials allowed | None (closed book) |
| Negative marking | No |
| Prerequisites | ISTQB CTFL v4.0 (or compatible Foundation Level) |

---

## ⭐ Course Structure

| Chapter | Topic | Weight | K-Levels |
|---------|-------|--------|----------|
| 1 | Introduction to Model-Based Testing | ~20% | K2 |
| 2 | MBT Modeling | ~25% | K2, K3 |
| 3 | Selection Criteria for Test Case Generation | ~25% | K2, K3 |
| 4 | MBT Test Execution and Analysis | ~15% | K2 |
| 5 | MBT Tools and Automation | ~15% | K2 |

**Key insight:** Chapters 2 and 3 are the highest-weighted and most technically demanding. Master state machines, decision tables, and coverage criteria.

---

## ⭐ How to Use This Course

1. Follow `STUDY_PLAN.md` — 2-3 weeks at 8-12 hours/week.
2. Read each chapter's `lessons.md` thoroughly.
3. Review chapter `README.md` for exam weighting.
4. Study `exam_traps.md`.
5. Complete all `practice_questions.md` — time yourself.
6. Work through all 3 case studies.
7. Attempt mock exams under timed conditions.
8. Review `EXAM_STRATEGY.md` in the final week.

---

## ⭐ Target Audience

- Testers seeking the ISTQB Model-Based Tester extension certification
- Testers using or planning to use model-based testing in their projects
- Test analysts and test engineers interested in MBT approaches

---

## ⭐ Prerequisites

- **Mandatory:** ISTQB CTFL v4.0 (Certified Tester Foundation Level)
- Recommended: Experience with state transition testing, decision tables, or formal modeling
- Recommended: Basic understanding of test design techniques (EP, BVA, state transition)
