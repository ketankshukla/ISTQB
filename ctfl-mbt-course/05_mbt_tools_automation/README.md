# Chapter 5 — MBT Tools and Automation

## Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand)
- **Focus:** MBT tool categories, selection criteria, automation integration

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Describe categories of MBT tools (model editors, test generators, execution frameworks)
2. Explain criteria for selecting MBT tools
3. Describe integration of MBT tools with test management and CI/CD
4. Describe the benefits and limitations of MBT tools

---

## Key Concepts

- Tool categories: model editors, test generators, execution frameworks, coverage analyzers
- Tool selection criteria: model notation support, integration, scalability, cost
- Integration with CI/CD, test management, and defect tracking
- Tool limitations and when manual MBT may be preferable
