# Chapter 1 — Introduction to Model-Based Testing

## Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2 (understand)
- **Focus:** MBT concepts, benefits, terminology, process overview

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Describe what model-based testing is and its purpose
2. Explain the benefits and limitations of MBT
3. Describe the MBT process (modeling, selection, generation, execution, analysis)
4. Identify when MBT is appropriate and when it is not
5. Define key MBT terminology

---

## Key Concepts

- MBT definition and purpose
- Benefits: systematic coverage, early defect detection, maintainability, traceability
- Limitations: modeling effort, model quality dependency, not suitable for all systems
- MBT process overview
- Relationship between models and requirements
