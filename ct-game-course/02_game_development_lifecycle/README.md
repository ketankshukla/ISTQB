# 🔁 Chapter 2 — Game Development Lifecycle

## 🎓 Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2, K3
- **Focus:** Milestones, alpha/beta/gold, agile in games, testing phases

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Describe game development milestones
2. Explain alpha, beta, and gold phases
3. Apply testing activities to each phase
4. Understand agile and waterfall in game development
5. Manage game builds and version control
6. Plan testing for live service games

---

## ⭐ Key Concepts

- Pre-production, production, post-production
- Milestones (vertical slice, alpha, beta, gold)
- Build verification testing
- Agile and iterative development
- Live service / games-as-a-service
- Version control for games
