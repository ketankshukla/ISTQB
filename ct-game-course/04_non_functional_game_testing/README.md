# Chapter 4 — Non-Functional Game Testing

## Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2, K3
- **Focus:** Performance, stability, compatibility, accessibility, localization

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Test game performance (FPS, loading times)
2. Verify stability (crash handling, memory leaks)
3. Test hardware and OS compatibility
4. Evaluate accessibility features
5. Validate localization quality
6. Test audio-visual synchronization

---

## Key Concepts

- Frame rate and rendering performance
- Loading time optimization
- Memory management and leaks
- Crash handling and recovery
- Hardware compatibility
- Accessibility testing
- Localization testing
