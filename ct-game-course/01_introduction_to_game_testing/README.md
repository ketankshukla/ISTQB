# 🌟 Chapter 1 — Introduction to Game Testing

## 🎓 Exam Weighting

- **Estimated weight:** ~10% (~4 questions out of 40)
- **K-levels:** K1, K2
- **Focus:** Game industry, testing roles, genres, player experience

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Describe the game industry landscape
2. Explain the role of testing in game development
3. Identify game genres and platform types
4. Describe player-centric testing approach
5. Recognize game-specific quality attributes

---

## ⭐ Key Concepts

- Game industry overview
- Testing roles (QA, compliance, localization)
- Game genres (RPG, FPS, puzzle, etc.)
- Platform types (console, PC, mobile, cloud)
- Player experience
- Game-specific quality attributes
