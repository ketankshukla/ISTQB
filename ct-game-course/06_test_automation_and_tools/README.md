# 🛠️ Chapter 6 — Test Automation and Tools

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2, K3
- **Focus:** Automated testing, test frameworks, CI/CD, analytics, bug tracking

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Select appropriate test automation for games
2. Use test frameworks and tools
3. Implement CI/CD for game builds
4. Apply analytics and telemetry testing
5. Use bug tracking effectively
6. Evaluate when to automate vs manual test

---

## ⭐ Key Concepts

- Automated gameplay testing
- Screenshot comparison
- Test frameworks (Unity Test Framework, Unreal Automation)
- CI/CD pipelines
- Telemetry and analytics
- Bug tracking workflows
- Manual vs automated balance
