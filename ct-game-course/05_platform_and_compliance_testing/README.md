# 🚀 Chapter 5 — Platform and Compliance Testing

## 🎓 Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2, K3
- **Focus:** Platform requirements, certification, ratings, submission process

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Describe platform-holder requirements
2. Explain certification process for each platform
3. Test for content ratings compliance
4. Verify online service integration
5. Test store and monetization features
6. Manage submission and release process

---

## ⭐ Key Concepts

- TRC, XR, Lotcheck
- Age/content ratings (ESRB, PEGI, CERO)
- Achievement/trophy systems
- Online services (PSN, Xbox Live, Nintendo Online)
- Store requirements
- Submission process
