# ⚙️ Chapter 4 — Test Automation in Agile

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Automation strategy, CI/CD, frameworks, maintainability

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Explain the role of test automation in agile
2. Describe automation strategies for different test levels
3. Explain CI/CD integration for test automation
4. Describe maintainability patterns for automated tests
5. Explain when and what to automate

---

## ⭐ Key Concepts

- Automation supports agile, does not replace manual testing
- Test pyramid applied to automation
- CI/CD pipelines and quality gates
- Automation frameworks and maintainability
- ROI of test automation
