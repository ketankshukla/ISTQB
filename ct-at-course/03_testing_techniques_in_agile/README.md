# 🎯 Chapter 3 — Testing Techniques in Agile

## 🎓 Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Exploratory testing, BDD, ATDD, TDD, test pyramid, testing quadrants

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Explain exploratory testing in agile
2. Describe Behavior-Driven Development (BDD)
3. Describe Acceptance Test-Driven Development (ATDD)
4. Describe Test-Driven Development (TDD)
5. Explain the test automation pyramid
6. Describe the agile testing quadrants

---

## ⭐ Key Concepts

- Exploratory testing: simultaneous learning, design, execution
- BDD: Given/When/Then, collaboration
- ATDD: Tests before implementation, acceptance level
- TDD: Unit level, red-green-refactor
- Test pyramid: unit → integration → UI
- Agile testing quadrants
