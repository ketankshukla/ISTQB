# Chapter 5 — Tools in Agile

## Exam Weighting

- **Estimated weight:** ~10% (~4 questions out of 40)
- **K-levels:** K2 (understand)
- **Focus:** Test management tools, CI/CD tools, collaboration tools, information radiators

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Explain the purpose of test management tools in agile
2. Describe CI/CD tools and their role in agile
3. Explain collaboration tools and information radiators
4. Describe tool selection criteria

---

## Key Concepts

- Test management tools in agile context
- CI/CD tools: Jenkins, GitHub Actions, GitLab CI
- Collaboration tools: Jira, Trello, Confluence

- Tool selection criteria
