# 📜 Chapter 1 — Agile Principles and Mindset

## 🎓 Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2 (understand)
- **Focus:** Agile Manifesto, Scrum, Kanban, XP, agile roles and ceremonies

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Explain the Agile Manifesto values and principles
2. Describe Scrum roles, ceremonies, and artifacts
3. Describe Kanban and its principles
4. Describe Extreme Programming (XP) practices
5. Explain the role of the tester in an agile team

---

## ⭐ Key Concepts

- Agile Manifesto: 4 values, 12 principles
- Scrum: roles (PO, SM, Dev Team), ceremonies, artifacts
- Kanban: visualization, WIP limits, flow
- XP: pair programming, TDD, CI, refactoring
- Tester's role in agile
