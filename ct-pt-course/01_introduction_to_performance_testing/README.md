# 🌟 Chapter 1 — Introduction to Performance Testing

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand)
- **Focus:** Definitions, objectives, performance characteristics, stakeholders

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Define performance testing and related terms
2. Explain the objectives of performance testing
3. Describe performance as a non-functional characteristic
4. Identify key stakeholders and their concerns
5. Understand the relationship between performance testing and other test types

---

## ⭐ Key Concepts

- Performance testing definition and scope
- Speed, scalability, and stability
- Non-functional testing characteristics
- Stakeholder perspectives (business, operations, development)
- Performance vs. functional testing
