# Chapter 2 — Types of Performance Testing

## Exam Weighting

- **Estimated weight:** ~20% (~8 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Load, stress, endurance, spike, volume, scalability, configuration, isolation testing

---

## Learning Objectives

By the end of this chapter, you should be able to:

1. Describe each type of performance test
2. Identify the appropriate test type for a given scenario
3. Explain the objectives and conditions for each test type
4. Understand the differences between test types
5. Select the right combination of test types for a project

---

## Key Concepts

- Load, stress, endurance (soak), spike, volume, scalability testing
- Configuration and isolation testing
- Test type selection based on objectives
- Combining multiple test types in a test strategy
