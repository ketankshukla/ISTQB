# 🛠️ Chapter 6 — Performance Testing Tools and Automation

## 🎓 Exam Weighting

- **Estimated weight:** ~15% (~6 questions out of 40)
- **K-levels:** K2 (understand), K3 (apply)
- **Focus:** Tool categories, scripting, CI/CD integration, cloud testing

---

## 🎯 Learning Objectives

By the end of this chapter, you should be able to:

1. Describe categories of performance testing tools
2. Explain scripting approaches and best practices
3. Describe CI/CD integration for performance testing
4. Understand cloud-based performance testing
5. Select appropriate tools for different scenarios

## ⭐ Key Concepts

- Commercial vs. open-source tools
- Recording vs. hand-coded scripts
- Parameterization and correlation
- Distributed load generation
- Performance gates in CI/CD
- Cloud-based testing benefits and challenges
